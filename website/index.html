Efficient Hybrid Query Execution Using Automatically-Generated Connectors

In this work we demonstrate an algorithm for the automated generation of connecting operators between arbitrary pairs of compatible database systems without the need to materialize the underlying data on the file system. Our approach improves performance and enables users to leverage functionality in external systems.
